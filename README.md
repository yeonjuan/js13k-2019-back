# Back Attacker
![screenshot](./big-400.png)  

My game for the js13kgames competition.  

## Story
This is the Ninja Forest.  
Invaders have entered!!.  
They have brought a gun cowardly.  
Get rid of them in the dark.  

## Scripts
* npm install
* npm run build
* npm run start

## How to Play

* Move and Attack: arrow key (↑, ↓, →, ←)
